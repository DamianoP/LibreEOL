Installazione

Ora verrà illustrato come installare il sistema partendo dai sorgenti PHP scaricati e successivamente installando il database.

Installazione dei sorgenti

Una volta creato e configurato il server web ed estratto il contenuto dell'archivio contenente i sorgenti di EOL occorrerà copiarli all'interno della root dell'host configurato.

ATTENZIONE : Per ragioni di sicurezza EOL elabora tutte le richieste attraverso un'unica pagina PHP chiamata index.php e situata all'interno della cartella public; pertanto, per poter utilizzare il sistema occorre che la root dell'host punti direttamente alla cartella public. Per fare questo basterà configurare correttamente la variabile DocumentRoot dell'host: ecco un esempio: 

    <VirtualHost *:80>
        ServerAdmin email@mio.dominio
        DocumentRoot "/var/www/html/eol/public"
        ServerName www.eol_mio.dominio
        ErrorLog "logs/eol/error_log"
        CustomLog "logs/eol/access_log" common
        <Directory>
            Options FollowSymLinks
            AllowOverride All
        </Directory/>
    </VirtualHost/>

Si ricorda di verificare che il proprietario dei file sorgenti di EOL sia l'utente che "lavora" sul server web (tipicamente si tratta dell'utente 'html' o 'apache'); se così non fosse si consiglia di modificare tale valore, controllando il manuale del vostro sistema operativo. Nei sistemi POSIX (come UNIX, Linux, OS/X etc) basta andare nella cartella principale di EOL e inviare il seguente comando da terminale:
$ chown -R apache:apache *

Similmente occorre verificare se l'utente che lavora sul server web abbia tutti i permessi necessari sui file sorgenti di EOL; se così non fosse si consiglia di modificare l'assegnazione dei permessi controllando il manuale del vostro sistema operativo. Nei sistemi POSIX (come UNIX, Linux, OS/X etc) basta andare nella cartella principale di EOL e inviare il seguente comando da terminale:
$ chmod -R 755 *.

Installazione del database

Una volta installati i sorgenti non rimane altro che creare il database su cui tutti i dati di EOL andranno memorizzati.

Il file DB MySQL.sql presente all'interno della cartella utils è uno script SQL che permette la creazione di un nuovo database MySQL all'interno del vostro server web. Questo script può essere utilizzato dal Command Line Tool di MySQL oppure attraverso uno dei numerosi strumenti di amministrazione MySQL presenti in rete, come phpMyAdmin, reperibile all'indirizzo http://www.phpmyadmin.net/home_page/index.php ma molto spesso già disponibile nei pacchetti (X)AMP.

Lo script genererà automaticamente un nuovo database chiamato 'eol', ma nel caso si voglia modificare tale nome basterà modificare il file DB MySQL.sql con un qualsiasi editor di testo e sostituire tutte le occorrenze di `eol` con `nome_del_db`, salvando il file al termine.

Quando il database sarà stato creato, per la sicurezza del vostro server web, si raccomanda di creare un nuovo utente protetto da password e di assegnare ad esso tutti i permessi di lavoro esclusivamente per il database dedicato ad EOL. Questa operazione è possibile attraverso phpMyAdmin (Guida) oppure con il Command Line Tool di MySQL (Guida).
