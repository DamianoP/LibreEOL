Requisiti per utilizzare EOL

Per installare EOL - Esami On Line occorre avere un web server totalmente funzionante.
Per un aiuto con l'installazione e la configurazione di un web server rimandiamo alle guide e ai manuali dei software che lo compongono.
L'attuale versione di EOL è stata testata per funzionare correttamente con i seguenti pacchetti:

  • Web server : Apache 2+ (http://httpd.apache.org)
  • PHP : PHP 5.3+ (http://www.php.net) con le librerie json e php-xml abilitate
  • DBMS : MySQL 5.5+ (http://www.mysql.com)

Nel caso si voglia installare EOL nella propria macchina è possibile usare uno dei pacchetti (X)AMP (Apache, MySQL, PHP) disponibili per il sistema operativo desiderato:

  • XAMPP - Multipiattaforma (http://www.apachefriends.org/en/xampp.html)
  • WAMP - Windows (http://www.wampserver.com/en/)
  • MAMP - Macintosh (http://www.mamp.info/en/index.php)

Recupero dei sorgenti

Andando direttamente nel repository EOL su GitHub e cliccando sul bottone Download Zip in basso a destra della pagina, sarà possibile scaricare tutto il software completo di EOL.
Terminato il download basterà estrarre l'archivio compresso con un qualsiasi estrattore tipo WinRAR, 7-Zip, etc...

Configurazione di Apache e PHP

Affinché EOL sia in grado di inviare correttamente e-mail occorre asicurarsi che il servizio sendmail sia attivo e funzionante.
Per i sistemi POSIX è sufficiente usare il comando seguente da terminale:
$ /usr/sbin/getsebool httpd_can_sendmail

Se il sistema risponde con un "on" allora il servizio e-mail è già attivo, altrimenti lo si può attivare usando il comando:
$ setsebool -P httpd_can_sendmail 1
Se si vuole verificare l'avvenuta attivazione basterà rieseguire il primo comando.

Per il corretto funzionamento di EOL è necessario che nella versione di PHP utilizzata siano abilitate le seguenti librerie:

  • JSON
  • php-xml

Inoltre è necessario che nel file php.ini di configurazione di PHP le vabiabili elencate siano impostate come segue:

  • short_open_tag = On
  • magic_quotes_gpc = Off
  • magic_quotes_runtime = Off
  • magic_quotes_sybase = Off
