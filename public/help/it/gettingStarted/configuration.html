Configurazione iniziale

Ora che tutto è stato installato correttamente non rimane altro che effettuare una prima configurazione: si rinomini il file config-sample.php contenuto nella cartella includes in config.php e si aggiorni il file con le impostazioni del proprio server web e del database creato:

$config['systemTitle'] = 'EOL - Esami On Line' Nome del sistema (visibile sul fondo di tutte le pagine)
$config['systemHome'] = 'http://www.eol_mio.dominio/' Homepage del sito web
$config['systemEmail'] = 'no-reply@eol_mio.dominio' Indirizzo e-mail dal quale inviare comunicazioni
$config['systemLang'] = 'en' Lingua di default del sistema (controllare la tabella Languages nel database)
$config['systemTimeZone'] = 'Europe/Rome' Time Zone di default del sistema (controllare la documentazione di PHP per la lista dei Time Zone)
   
$config['dbHost'] = 'localhost' Host del database (tipicamente 'localhost')
$config['dbPort'] = '3306' Porta d'accesso al database (porta di default di MySQL '3306')
$config['dbName'] = 'eol' Nome del database creato utilizzando lo script (nome di default 'eol')
$config['dbUsername'] = 'username' Utente con accessi totali al database di EOL
$config['dbPassword'] = 'password' Password dell'utente con accessi totali al database di EOL

È possibile, ma non necessario, configurare anche la lingua di default per i plug-in CKEditor e Roxy Fileman:

  • CKEditor: aprire il file config.js presente nella cartella public/ckeditor e modificare la stringa
    config.defaultLanguage = 'en'
    usando una delle lingue disponibili nella cartella public/ckeditor/lang (ad esempio usare 'en' per l'inglese, 'it' per l'italiano, etc... ).
  • Roxy Fileman: aprire il file conf.json presente nella cartella public/fileman e modificare la stringa
    "LANG" : ""
    usando una delle lingue disponibili nella cartella public/fileman/lang (ad esempio usare 'en' per l'inglese, 'fr' per il francese, etc...).
